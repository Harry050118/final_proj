**Agentic Retrieval-Augmented Generation.** Retrieval-augmented generation (RAG) enhances large language models by grounding generation in external knowledge, mitigating hallucinations and improving factual accuracy on knowledge-intensive tasks [1]. Early RAG systems typically perform a single round of retrieval based on the initial query, which limits their effectiveness on complex questions requiring multi-step reasoning [1, 2]. To address this limitation, agentic RAG frameworks have emerged, enabling models to interleave reasoning with dynamic, multi-turn retrieval [2, 3]. Search-o1 introduces an agentic workflow that triggers iterative document retrieval when uncertainty is detected and employs a dedicated reasoning module to analyze retrieved content, demonstrating gains across science, math, coding, and open-domain question answering [3]. IRCoT proposes interleaving retrieval with chain-of-thought reasoning, where each reasoning step guides the next retrieval and vice versa, improving multi-hop QA performance without requiring task-specific training [2]. These prompting-based approaches highlight the potential of agentic search but rely on frozen model capabilities rather than learned search policies.

**Reinforcement Learning for Search Agents.** Recent work has turned to reinforcement learning (RL) to train LLMs to autonomously decide when and what to retrieve during reasoning. Search-R1 introduces an RL framework that enables models to generate search queries during step-by-step reasoning, using outcome-based rewards and retrieved token masking for stable training, achieving substantial gains across seven QA benchmarks [4]. R1-Searcher proposes a two-stage outcome-based RL method that first teaches retrieval invocation patterns and then refines answer generation, outperforming strong baselines on multi-hop QA [5]. R1-Searcher++ extends this with a memory mechanism and explicit incentives for using internal knowledge when confident, reducing retrieval counts while maintaining accuracy [6]. s3 demonstrates that a standalone search agent can be trained with minimal data by decoupling search from generation and using a reward that measures improvement over naive retrieval [7]. Multi-agent RL approaches have also been explored, treating components such as query rewriting and document selection as cooperative agents jointly optimized toward end-to-end accuracy [8]. These methods establish RL as a powerful paradigm for training search agents but predominantly rely on coarse, outcome-level rewards that provide limited guidance on the quality of intermediate search decisions.

**Efficiency and Suboptimal Search Behaviors.** A critical challenge in agentic RAG is the prevalence of suboptimal search behaviors, particularly over-search (redundant or unnecessary retrieval) and under-search (failure to retrieve when needed), which degrade both efficiency and reliability [9, 10]. Search Wisely identifies these behaviors as linked to model uncertainty about knowledge boundaries and proposes beta-GRPO, which uses confidence thresholds derived from token probabilities to reward high-certainty search decisions, reducing both over-search and under-search rates [9]. AutoSearch introduces a self-answer mechanism that dynamically determines minimal sufficient search depth, rewarding attainment of that depth while penalizing over-searching [10]. RAG-R1 explores multi-query parallelism to reduce retrieval rounds and inference time by exploring diverse knowledge paths simultaneously [11]. Cost-aware approaches have also sought to balance retrieval depth with computational expense [12]. Despite these advances, existing methods typically rely on heuristics such as length penalties or confidence thresholds that can oversimplify the problem or misjudge when retrieval is truly necessary [9].

**Process-Level Supervision in Reasoning.** Beyond agentic RAG, the broader literature on RL for LLM reasoning has increasingly emphasized the value of process-level supervision over sparse outcome rewards. Process Reinforcement through Implicit Rewards (PRIME) introduces an implicit process reward modeling framework that enables online RL using only outcome labels, achieving significant gains on reasoning benchmarks while reducing data requirements [13]. PRL decomposes entropy-regularized RL objectives into intermediate process rewards derived from policy entropy ratios, providing theoretically grounded dense supervision during reasoning [14]. Search-P1 proposes path-centric reward shaping that evaluates reasoning trajectories at the path level using dual-track scoring and soft outcome assignment, improving sample efficiency and convergence [15]. DecEx-RAG introduces a process-supervised MDP framework for agentic RAG that decomposes decision and execution stages and prunes redundant search branches using aggregated rollout rewards [16]. Beyond Correctness embeds fine-grained faithfulness metrics into RL training to ensure intermediate reasoning steps align with retrieved evidence, revealing a disconnect between final correctness and reasoning validity [17]. These works collectively demonstrate that optimizing the reasoning process itself, rather than only the final outcome, can yield more robust and efficient agent behavior.
